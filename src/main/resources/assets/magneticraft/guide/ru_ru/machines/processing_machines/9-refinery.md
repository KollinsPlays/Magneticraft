# Очистительный завод


Эта высокая башня может отделить масло в разные жидкости, исходя из их плотности.

На заводе требуется пар для работы, а скорость зависит от рецепта.

На первом уровне: вход жидкости находится сзади, а вход пара находится сбоку.
Остальные уровни имеют выходы с каждой стороны, а уровень указывает тип вывода.

This structure is a [multiblock](../../3-multiblocks.md) of size *3, 9, 3* (x, y, z).
