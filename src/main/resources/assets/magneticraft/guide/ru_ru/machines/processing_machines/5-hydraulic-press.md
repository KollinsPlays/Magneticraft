# Гидравлический пресс

Эта машина разбивает металлы для создания пластин. 
Он имеет 3 режима, нормальные плиты, легкие плиты и тяжелые плиты.

Потребляет 60 ватт на максимальной мощности

This structure is a [multiblock](../../3-multiblocks.md) of size *3, 5, 3* (x, y, z).

Materials needed:
- 1 x Макет гидравлического пресса
- 14 x Машинный Блок (Железная Решётка)
- 7 x Колона Поддержки
- 2 x Машинный Блок (Медная Катушка)
- 1 x Машинный Блок (Полосатый)
- 2 x Машинный Блок (Рифленое Железо)
